プロジェクト概要

Replace-Soloは、プライバシーを最優先に設計された、ブラウザ内で処理が完結するMicrosoft Loop専用の置換ツールです。 標準機能では提供されていない一括置換機能を、Microsoft Loopのエディタ上でシームレスに提供します。

特徴

Loop特化

Microsoft Loopのエディタ構造を解析し、安全かつ確実にテキストを置換します。

完全ローカル実行

形態素解析を含め、すべての処理をブラウザ内で行います。データが外部サーバーに送信されることはありません。

トラッキングなし

利用状況の追跡や分析ツールは一切含まれていません。プライバシーを最優先しています。

Material 3 UI

清潔感のあるMaterial 3デザインを採用し、直感的で分かりやすい操作を提供します。

Copilot用プロンプト生成

蓄積された辞書データを活用し、Microsoft 365 Copilot（@facilitator）に聞き間違いの修正を指示するためのプロンプトをワンクリックで生成します。

インストール方法

  1. Chrome ウェブストアにアクセスします。
  2. 「Chromeに追加」ボタンをクリックします。
  3. 拡張機能の固定をオンにすると、アクセスしやすくなります。
