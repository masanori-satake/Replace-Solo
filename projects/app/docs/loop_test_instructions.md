# Loop テスト用ページ作成指示書

この指示書は、Replace-Solo の「入力エミュレーション」および「対象抽出」の精度を検証するためのテストページを Microsoft Loop 上に作成するためのものです。

## 目的
Loop の様々なエディタ要素（リッチテキスト、メンション、コンポーネント等）に対して、Replace-Solo が正しくテキストを抽出し、置換を実行できるかを確認します。

## 作成手順

### 1. 新しい Loop ページの作成
Loop ワークスペースで新しいページを作成し、タイトルを「Replace-Solo Test Page」としてください。

### 2. 基本テキスト要素の配置
以下の内容をページにコピー＆ペースト、または手入力してください。

*   **見出し1**: プロジェクト概要
*   **見出し2**: 昨日の会議要約
*   **見出し3**: 決定事項
*   **見出し4**: 補足情報
*   **通常の段落**: これは Replace-Solo のテスト用の文章です。形態素解析が正しく行われるか確認します。
*   **太字・斜体**: **重要事項**を*斜体*で強調します。

### 3. リスト要素の配置
以下のリストを作成してください。

*   **箇条書きリスト**:
    *   タスク A
    *   タスク B
*   **段落番号リスト**:
    1.  手順 1
    2.  手順 2
*   **チェックリスト**:
    *   [ ] 未完了タスク
    *   [x] 完了済みタスク

### 4. 特殊要素の配置（重要）
Loop 特有の要素を配置します。

*   **ユーザメンション**: `@` を入力し、自分自身やチームメンバーをメンションしてください。
*   **日付**: `/date` と入力し、今日の日付を挿入してください。
*   **ラベル**: `/label` と入力し、「進行中」や「完了」などのラベルを挿入してください。

### 5. コンポーネントの配置
*   **表**: `/table` で表を作成し、セル内に適当な日本語（例：「担当者」「期限」）を入力してください。

---

## 検証方法

1.  作成したページを開いた状態で、Replace-Solo のサイドパネルから「対象抽出」をクリックします。
2.  「デバッグ」タブ（バージョン番号を3回クリックして表示）から「抽出結果の一括ダウンロード」を行います。
3.  ダウンロードされた JSON ファイルを開き、以下の点を確認します。
    *   `text`: ページ内のテキストが漏れなく抽出されているか。
    *   `extractedWords`: 見出しやリスト内の単語、メンションされた名前などが正しく抽出されているか。
    *   `tokens`: kuromoji による品詞分解が期待通りか。
4.  「入力エミュレーションを使用」をオンにした状態で、いくつかの単語を置換し、Loop 画面上の表示が崩れずに書き換わるかを確認します。
