# Replace-Solo 機能仕様書 (docs/spec.md)

## 1. プロジェクト概要
- **名称**: Replace-Solo（リプレイス・ソロ）
- **サブタイトル**: 〜データがブラウザの外に出ない、自分専用の固有名詞・一括置換ツール〜
- **目的**: Microsoft Loop, Google Docs 等のリッチエディタ上で、議事録等の固有名詞や特定の口癖を抽出し、一括置換する。

## 2. 主要機能

### A. 解析・抽出 (Extraction)
- **トリガー**: サイドパネル上の「解析実行」ボタン押下。
- **対象**: アクティブなタブのページ全体。
- **ロジック**: `kuromoji.js` を使用し、名詞（特に固有名詞）を中心に抽出。
- **リスト表示**: 重複する単語は1行に集約。

### B. 置換エンジン (Replacement)
- **置換モード**:
  1. `document.execCommand('insertText')` (入力エミュレーション): リッチエディタの履歴やカーソル位置を維持。
  2. DOM直接書き換え: シンプルなテキスト置換。
- **自動選択**: `loop.microsoft.com`, `docs.google.com` 等の特定ドメインでは「入力エミュレーション」をデフォルト。
- **手動切替**: UI上でユーザーがモードを切り替え可能。

### C. 辞書データ管理 (Dictionary)
- **構造**: `{"置換候補文字": ["元単語1", "元単語2", ...]}` (1:N 形式)
- **永続化**: `chrome.storage.local` を使用。
- **インポート/エクスポート**: JSON形式。上書き/追加を選択可能。スキーマバリデーションを実装。

## 3. UI詳細 (サイドパネル)

### 抽出リスト・操作テーブル
| 要素 | 種別 | 動的挙動 |
| :--- | :--- | :--- |
| 適用 | M3 Checkbox | 初期値：辞書マッチならON、なければOFF。候補入力で自動ON。候補が空なら「削除」扱い。 |
| 元単語 | Body Large | 抽出された単語。 |
| 置換候補 | M3 Filled Text Field / Select | 辞書から自動展開（プルダウン選択可能）。自由入力時は辞書登録ON。 |
| 辞書登録 | M3 Checkbox | 実行時に保存するか。新規入力で自動ON、辞書既知ならDisable、空なら自動OFF。 |
| 単体実行 | M3 IconButton | その行のみ即座に置換。 |

### 操作エリア
- **手動追加**: 元単語をリストに直接追加。
- **一括置換**: 「適用」がONの項目を全実行。
- **リセット**: 解析直後の状態に戻す。

## 4. 権限 (Permissions)
- `sidePanel`: サイドパネルの利用。
- `storage`: 辞書データの保存。
- `scripting`: ページ内での置換実行。
- `activeTab`: 現在のタブへのアクセス。

## 5. 対象ドメイン (自動判定)
- `loop.microsoft.com`
- `docs.google.com`
- `sheets.google.com`
- その他 Microsoft オンラインエディタ
