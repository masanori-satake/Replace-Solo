# AGENTS.md

## プロジェクト設計方針 (Design Policy)

- **コンセプト**: 「完全ローカル実行」による究極のプライバシー保護。
- **デザイン指針**: Google Material 3 (M3) をベースとし、QuickLog-Solo との親和性と統一感を最優先する。清潔感、適切な余白、直感的なアイコン配置を重視。
- **技術スタック制限**:
  - 外部OSSライブラリの使用を極力排除する（`kuromoji.js` を除く）。
  - UIコンポーネントは外部フレームワーク（React, Vue, Tailwind CSS等）を使わず、純粋なHTML/CSS (Material 3 準拠) で実装する。
  - 依存関係の脆弱性リスクを最小化し、長期的な保守性を確保する。

## 開発ガイドライン

- **プライバシー第一**: 全ての処理はブラウザ内で完結させ、外部サーバーへのデータ送信は一切行わない。
- **セキュリティ**: Chrome Extension Manifest V3 に準拠。スクリプトの実行は最小限の権限で行う。
- **コード品質**: 読みやすく、拡張しやすいプレーンなJavaScript（Vanilla JS）を記述する。

## プロジェクト構造

- `manifest.json`: 拡張機能の設定
- `sidepanel.html/js/css`: サイドパネルのUIとロジック
- `content.js`: ページ内のテキスト抽出および置換の実行
- `background.js`: 拡張機能のライフサイクル管理、サイドパネルの制御
- `lib/kuromoji/`: 形態素解析ライブラリと辞書データ
- `docs/spec.md`: 詳細な機能仕様
- `AGENTS.md`: 設計ポリシーおよび本ドキュメント
